# monte-Carlo-Simulation
Dieses Tool simuliert zukünftige Aktienkursverläufe mittels geometrischer Brownscher Bewegung (GBM) oder Merton-Jump-Diffusion. Es liefert eine umfassende Risikoanalyse direkt im Terminal – inklusive Endpreisverteilung, Zeithorizont-Renditen, VaR/CVaR, maximalen Drawdowns und einem visuellen ASCII-Histogramm.
